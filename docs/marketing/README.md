## What you must do yourself

These items cannot be generated as files in the repo:

1. **Record and edit video** — TikTok/Reels/Shorts clips, GIFs, trailer (scripts are ready; footage is not).
2. **Capture marketing screenshots** — Follow the nine-shot list in `press-kit-checklist.md` and `marketing-handoff.md`.
3. **Steam / store art** — Capsule (1232×706), header (920×430), library hero, screenshots at 1920×1080 or 16:9.
4. **Create accounts** — TikTok, YouTube, Twitter/X, Reddit, Discord server, Steamworks (if you go to Steam).
5. **Post and engage** — Publish drafts, reply to comments, run AMA, send influencer emails.
6. **Pay Steamworks** — $100 USD app fee per game (recoupable after ~$1,000 in Steam revenue).
7. **Legal / business** — Tax forms, privacy policy if you collect analytics, trademark search on “Boss Rush” if branding widely.
8. **Paid ads** — Optional; kit assumes organic + micro-creator first (see market check).
